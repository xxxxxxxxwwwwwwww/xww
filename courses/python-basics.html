模块4：控制流与循环结构

学习Python中的条件语句和循环结构，掌握程序的流程控制。

条件语句

Python中的条件语句使用 if、elif 和 else 关键字。

# 条件语句示例
age = 18
if age >= 18:
    print("你是成年人")
elif age >= 13:
    print("你是青少年")
else:
    print("你是儿童")

循环语句

Python支持 for 循环和 while 循环。

# for循环示例
for i in range(5):
    print(i)

# while循环示例
count = 0
while count < 5:
    print(count)
    count += 1
