# Xww的个人课程页面

这是一个纯静态的个人课程信息展示页面，用于展示广东科学技术职业学院商学院商务数据分析与应用专业学生Xww的课程信息。

## 功能特点

- **个人信息展示**：显示学生姓名、学校、专业等基本信息
- **课程列表**：展示多个课程卡片，包含课程名称和简短描述
- **课程详情**：点击课程卡片可展开查看课程详细信息
- **响应式设计**：适配桌面、平板和移动设备
- **现代化界面**：使用Tailwind CSS构建，美观且专业

## 技术栈

- **前端**：纯静态HTML + CSS + JavaScript
- **样式框架**：Tailwind CSS
- **图标**：Font Awesome
- **部署平台**：Cloudflare Pages

## 如何部署

### 方法一：Cloudflare Pages

1. 在GitHub上创建一个新的仓库
2. 将本项目的文件上传到仓库
3. 登录Cloudflare控制台，导航到Pages
4. 点击"创建项目"，选择GitHub作为源
5. 选择你的仓库，配置构建设置：
   - 构建命令：无需构建命令（纯静态文件）
   - 构建输出目录：/（根目录）
6. 点击"部署"，等待部署完成
7. 部署完成后，你将获得一个Cloudflare Pages的域名

### 方法二：本地预览

1. 克隆或下载本项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 或者使用本地服务器（如VS Code的Live Server插件）预览

## 如何更新内容

1. 打开 `index.html` 文件
2. 修改个人信息部分：
   - 姓名、学校、专业等基本信息
3. 修改课程信息部分：
   - 课程名称、描述、详情等
4. 保存文件并重新部署
